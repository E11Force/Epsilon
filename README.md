**Epsilon** - is a competitive open-source chess engine created by [**Fozik**](https://github.com/SimplyFozik) using **C++** as main language

> [!NOTE]
> This engine is on *alpha* testing and still being under development

### Usage
this engine compatible with UCI protocol, so to use it you might need ***chess interface*** for ex. you may use [**Cute Chess**](https://github.com/cutechess/cutechess), or [**En Croissant**](https://encroissant.org)
