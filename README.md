  <h1>Epsilon</h1>

Epsilon - is a competitive open-source chess engine created by [**Fozik**](https://github.com/SimplyFozik) using C++ as the main language.

> [!NOTE]

> This engine is on alpha testing and still being under development

### Usage

This engine is compatible with the UCI protocol, so to use it, you might need ***a chess interface***. For example, you may use [**Cute Chess**](https://github.com/cutechess/cutechess), or [**En Croissant**](https://encroissant.org)
